# Complete

Finalize and wrap up the current work.

## Process
1. Run `/verify` to confirm all checks pass
2. Run `/review` for a final code review
3. Summarize what was done:
   - Changes made (files, features)
   - Decisions taken (link to ADRs if any)
   - Known limitations or follow-ups
4. Suggest next steps

## Rules
- Do not mark complete if verify fails
- Be honest about limitations and tech debt
