# Research

Conduct deep research and produce a findings document.

## Process
1. Clarify the research question
2. Search the codebase, documentation, and web as needed
3. Synthesize findings into a report:
   - Summary / answer
   - Evidence and sources
   - Alternatives considered
   - Recommendation
4. Save to `.claude/plans/research-[topic].md`

## Rules
- Cite sources (file paths, URLs, documentation sections)
- Distinguish facts from opinions/recommendations
- Note confidence level for each finding
